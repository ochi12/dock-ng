# Dock Next Generation (DOCK!NG)
A lightweight non-fixed dock support for GNOME Shell using the native dash with simple but adaptive intellihide. 

[<img width="256" alt="get-it-on-eog" src="https://github.com/user-attachments/assets/28a16af9-847b-449a-afdc-ae6511f6f692" />](https://extensions.gnome.org/extension/8682/dockng/)

## Features
* Lightweight and seamlessly integrated with the native dash
* Simple intellihide that adapts based on the focused app
* Smooth, native-style reveal and hide animations
* Multimonitor and adaptive monitor stack support (lacks testing â€” may behave unexpectedly)

## Installation
For normal users, we recommend installing the extension from  https://extensions.gnome.org/extension/8682/dockng/
<br>
<br>
If you are a contributor or just prefers installing with the latest changes:
<br>
<br>
**Clone this repository**
```bash
$ git clone https://github.com/ochi12/dock-ng#
```
**Run build and install at once** - assuming you have `Make` installed in your system
```bash
$ make all
```
